# ✋ Hand Gesture Controlled Presentation System

A Python-based project that uses computer vision to control PowerPoint slides using hand gestures via webcam.

---

## 🚀 Features

* Control slides (Next / Previous) using gestures
* Draw and annotate on slides
* Virtual keyboard input
* Real-time hand tracking

---

## 🛠️ Tech Stack

* Python
* OpenCV
* MediaPipe
* cvzone
* PyAutoGUI

---

## ⚙️ Setup & Installation

### 1. Clone Repository

git clone https://github.com/rickyparmar/Hand-Gesture-Presentation.git

### 2. Go to Folder

cd Hand-Gesture-Presentation

### 3. Install Dependencies

pip install opencv-python mediapipe cvzone pyautogui

### 4. Run Project

python main.py

---

## 💡 Usage

Use your hand gestures in front of the webcam to control slides and interact with the presentation.
